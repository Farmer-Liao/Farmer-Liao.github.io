Issue Admin Commands

Last updated: 2023/10/29

Namespace nvme::admc

The functions which are devised to issue Admin commands are called with nvme::admc namespace resolution.

Function Handle

To issue an Admin command to the NVMe SSD, the function (PCI function; either physical function or virtual function) handle should be specified.

Command Handle

After the function has issued an Admin command to the NVMe SSD, a command handle is returned by the function.

pspp()

The pspp() is used to poll the status of the command issued. The command handle is passed to pspp() function. The pspp() function returns a 64-bit value with:
(1) command completion flag
(2) command timeout flag
(3) status field of the completion queu entry
(4) command specific field of the completion queue entry

Example

This example shows how to issue an identify command to retrieve the Identify Controller data structure. 

      
    funcHandle = nvme::getFuncHandle(EC, 0);
    nvme::init(EC, funcHandle);
    bufHandle = nvme::ddbAlloc(EC, funcHandle, 4096);
    csn = 1;
    nsid = 0;
    CNTLID = 0;

    do 
        cmdHandle = nvme::admc::identify(EC, funcHandle, cns, nsid, CNTLID, bufHandle);
    while (cmdHandle == 0);

    while (1) {
        stsVal = nvme::pspp(cmdHandle);
        if (stsVal)
            break;
    }

    if (stsVal & nvme::cmdCplFlag) {
        nvme::ddbDump(EC, bufHandle, 0, 0);
    }
    else {
        ...
    }
